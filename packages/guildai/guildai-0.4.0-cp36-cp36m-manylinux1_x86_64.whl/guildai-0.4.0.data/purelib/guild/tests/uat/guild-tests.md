# Guild tests

Guild tests can be run with the `check` command using the `-T` option
(`-n` here skips the general check info as we're just interested in
tests).

    >>> run("guild check -nT", ignore="FutureWarning") # doctest: +REPORT_UDIFF
    internal tests:
      config:                  ok
